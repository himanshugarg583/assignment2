# EG TRANS Logistics Website

Premium logistics marketing site built with React, Vite, Tailwind CSS, Framer Motion, GSAP, and Lenis smooth scrolling.

## Getting Started

1. Install dependencies
   npm install

2. Start dev server
   npm run dev

3. Build for production
   npm run build

## Notes
- Uses Google Fonts (Sora and Manrope).
- Graphics are lightweight SVG/shape-based placeholders.
